# Task 4 - Common Network Security Threats

## Objective

The objective of this task is to study common network security threats, understand how they affect systems and networks, and identify best practices to prevent them.

---

# Introduction

Network security threats are attacks or malicious activities that target computer networks to steal data, disrupt services, or gain unauthorized access. Organizations use firewalls, intrusion detection systems, encryption, and security policies to reduce these risks.

---

# Common Network Security Threats

## 1. Malware

Malware is malicious software designed to damage, disrupt, or gain unauthorized access to computer systems.

Examples:
- Virus
- Worm
- Trojan Horse
- Ransomware
- Spyware

Prevention:
- Install antivirus software
- Keep software updated
- Avoid downloading unknown files

---

## 2. Phishing

Phishing is a social engineering attack that tricks users into revealing sensitive information such as passwords or banking details.

Prevention:
- Verify email senders
- Avoid suspicious links
- Enable Multi-Factor Authentication (MFA)

---

## 3. SQL Injection

SQL Injection occurs when attackers insert malicious SQL code into application input fields to access or manipulate databases.

Prevention:
- Prepared Statements
- Parameterized Queries
- Input Validation

---

## 4. Denial of Service (DoS)

A Denial of Service attack floods a server with traffic, making services unavailable.

Prevention:
- Firewalls
- Rate Limiting
- Traffic Filtering

---

## 5. Distributed Denial of Service (DDoS)

A DDoS attack uses multiple compromised systems to overload a target server.

Prevention:
- DDoS Protection Services
- Load Balancing
- CDN

---

## 6. Man-in-the-Middle (MITM)

Attackers secretly intercept communication between two parties.

Prevention:
- HTTPS
- VPN
- Encryption

---

## 7. Password Attacks

Attackers attempt to guess or steal passwords.

Examples:
- Brute Force
- Dictionary Attack
- Credential Stuffing

Prevention:
- Strong Passwords
- MFA
- Password Managers

---

## 8. Insider Threats

Employees or trusted users intentionally or accidentally compromise security.

Prevention:
- Access Control
- User Monitoring
- Security Awareness Training

---

## Best Security Practices

- Keep systems updated.
- Use strong passwords.
- Enable Multi-Factor Authentication.
- Regularly back up important data.
- Install firewalls and antivirus software.
- Monitor network activity.
- Educate users about cybersecurity.

---

# Conclusion

Understanding common network security threats helps organizations protect sensitive information and maintain secure systems. Implementing security best practices significantly reduces the risk of cyber attacks.
