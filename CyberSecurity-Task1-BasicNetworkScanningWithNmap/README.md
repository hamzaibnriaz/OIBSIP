
## Nmap Installation

### Operating System
Kali Linux (VMware)

### Installation

Kali Linux comes with Nmap pre-installed.

To verify the installation, the following command was used:

```bash
sudo apt install nmap
```

Output:

```
nmap is already the newest version (7.99+dfsg-1kali1).
nmap set to manually installed.
```

The installed version was confirmed using:

```bash
nmap --version
```
